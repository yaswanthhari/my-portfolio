# Real-time File Organizer

Automatically organizes a watched folder (e.g., Downloads) into subdirectories based on file extensions.

## Features
- 🚀 Real-time monitoring with Watchdog
- 📝 Production-grade logging with rotation
- ⚙️ YAML configuration

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Update `config.yaml` to point to your desired directory.
3. Run: `python organizer.py`
